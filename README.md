## 1. Project Overview
SmartVet is a specialized IoLT (Internet of Livestock Things) platform utilizing passive NFC technology to identify and manage:
* **Pet Segment:** Companion animal safety and medical history tracking.
* **Agro Segment:** Livestock identification compliant with ARiMR regulations.

**Key Differentiator:** Battery-free (Always-On) operation vs. traditional GPS trackers.

## 2. Technical Scope
* **Core:** NFC Passive Tag Communication Protocols.
* **Integrations:** Klinika XP, Vetfile, Safe-Animal.
* **Infrastructure:** Secure Cloud Backend & Mobile Interface.
